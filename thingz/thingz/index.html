    Thingz¶

    La Carte galaxia a été conçu par Thingz, une entreprise constituée de plein de gens sympathiques !

    Pour réaliser cette carte, ils ont assemblé plusieurs composants électroniques qui permettent d'ajouter à la carte des capacités, comme capter la luminosité, le magnétisme, l'accélération, de jouer du son, etc.

    Pour qu'on puisse utiliser ces fonctionnalités, il a fallu que Thingz créé un code qui traduit une utilisation simple en un code compréhensible par chacun des composants. Ils ont regroupé tout ce code dans une bibliothèque (également appelé librairie ou module) et l'ont appelé thingz (pour qu'on se le rappelle).

    Dès lors, si vous importez la bibliothèque thingz au début de votre programme alors vous pourrez appeler une fonction simple d'un composant.

    1
2
    # Ici, j'importe toute la bibliothèque.
import thingz

    Ou

    1
2
3
    # J'importe tous les modules depuis le module thingz
# * est un joker qui signifie "tout"
from thingz import *
